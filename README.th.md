# 🌍 ต้องการการแปล!

> **คุณพูดภาษาไทยไหม?** เราต้องการคุณ!

ไฟล์นี้ต้องการการแปลจากภาษาอังกฤษเป็นภาษาไทย

## วิธีการมีส่วนร่วม

1. **Fork** repository นี้
2. **แปล** จาก [README.md](README.md)
3. เปิด **Pull Request**

### รางวัล:
- 🎖️ เหรียญนักแปล
- ⭐ +50 คะแนน karma
- 💜 ความขอบคุณชั่วนิรันดร์
